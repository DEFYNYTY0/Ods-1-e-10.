Como foi produzido esse site?

Com os conhecimentos adquiridos pelos instrutores do Senac, decidimos fazer um site para esse trabalho, contendo vídeos explicativos sobre as ODS e tudo do que nós aprendemos até agora.

1° Abrimos a estrutura básica do HTML colocando um ponto de exclamação.

2° Utilizamos tags como DIV e HEADER para separar os conjuntos do site.

3° Usamos a tag VIDEO para colocar os videos no site.

4° Usamos âncoras nos links acima do site para navegar entre as paginas.

O que é ODS?

Os ODS – Objetivos de Desenvolvimento Sustentável da ONU – são um pacto global firmado em 2015 para que mais ninguém seja deixado para trás.
